# INSTALL AND RUN DEV VERSION

clone the git repository, then

```
yarn install
yarn start
```

See presentationals components in the storybook

```
yarn storybook
```

# DEPLOY

Create the build with

```
yarn build
```

Deploy on now

```
yarn deploy
```
