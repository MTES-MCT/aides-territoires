Porteurs d’aides masqués

Cette page liste les porteurs d’aide correpondant aux paramètres de votre structure.

Vous pouvez les masquer individuellement dans les résultats de recherche.

{% if nb_excluded %} {{ nb_excluded }} porteur{{ nb_excluded|pluralize }} masqué{{ nb_excluded|pluralize }}. {% else %} Tous les porteurs sont affichés. {% endif %}

Vous souhaitez masquer certains porteurs de toutes vos recherches ? vous pouvez le faire sur cette page. Cette opération s’appliquera à toutes vos recherches, mais elle est parfaitement réversible : vous pouvez changer d’avis à tout moment selon vos besoins, ou même faire varier les paramètres propres à une recherche particulière, afin de ne louper aucune aide !

Le choix de masquer un porteur d’aides est individuel, et ne sera pas partagé avec les membres de votre équipe.

Néanmoins, si un membre de votre équipe repère une aide d’un porteur que vous avez masqué et l’intègre à votre projet, vous serez en mesure de voir cette aide.
